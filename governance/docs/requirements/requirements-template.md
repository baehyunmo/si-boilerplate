## 2. 기능 요구사항 (Functional Requirements)

### 2.1 요구사항 분류 체계

- **FR-1xx**: 사용자 관리
- **FR-2xx**: 핵심 업무 기능
- **FR-3xx**: 데이터 관리
- **FR-4xx**: 연동/인터페이스
- **FR-5xx**: 보고/통계
- **FR-9xx**: 관리자 기능

> **우선순위 기준**
> - **필수 (Must)**: 시스템 운영에 반드시 필요한 기능

> **작성 가이드**
> 1. 요구사항 ID는 분류 체계에 따라 일관되게 부여한다.

> 3. 우선순위는 고객과 합의하여 결정한다.
